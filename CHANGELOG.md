# Changelog

All notable changes to `laravel-toasts` will be documented in this file.

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- Toast notification system with multiple types (success, error, warning, info, question, loader)
- Emoji support for toast notifications
- Configurable position (top-right, top-left, bottom-right, bottom-left, top-center, bottom-center)
- Customizable duration and animations
- Confirmation toasts with callbacks
- Artisan command to inject toast view into layouts
- Livewire integration support
- JavaScript API for client-side toast management
- Comprehensive helper functions
- FontAwesome icon support
- RTL language support (Arabic)

### Features
- Chainable toast builder pattern
- Session-based toast persistence
- Automatic layout injection
- Asset publishing with individual tags
- Multiple color themes
- Configurable animations (fade, slide, bounce, zoom)
